# now we need to find the optimal number of clusters.

# We will use 'Within Clusters Sum of Squares' (WCSS)

# Finding WCSS for different number of clusters using a for loop.

# Here, our for loop is going to get the inertia for K = 1 all the way to 10.

# It saves the inertia into a list wcss

# This inertia is vital in determining how many clusters are optimal.

wcss = []
for i in range(1,11):
        kmeans = KMeans(n_clusters = i, init = 'k-means++', random_state = 42)
        kmeans.fit(X)
        wcss.append(kmeans.inertia_)
