What is Sentiment Analysis?

Sentiment analysis is a technique that uses natural language processing to extract and classify the emotions or opinions expressed in a text. For example, if someone writes a product review, sentiment analysis can tell us whether the review is positive, negative or neutral. Sentiment analysis can be useful for businesses to understand customer feedback, for researchers to analyze social media posts, or for anyone who wants to explore the feelings and attitudes of people towards a topic.

How I carried out the Project

Step 1:ETL
First, I obtained the data needed and wrote a function that loaded the texts and their labels into pandas dataframes(train and test sets).
Step 2:Data Pre-Processing
I proceeded to normalize the text data. This involves removing capitalizations, punctuation marks, etc. This is necessary for Machine Learning algorithms to be able to read the texts without problems.To do this I created a for loop that inputs every comment and put it in lower case, and then remove all other special characters and punctuation marks.
Step 3: Model Training
To train the model I first used a Count Vectorizer to transform the texts into a matrix counts on which I then trained a Logistic Regression algorithm after doing a train/test split on the data. This algorithm is great for binomial variables and in this case, I only considered whether a comment is positive or not so it was feasible to use it.
Step 4: Model Evaluation and Testing
To test the algorithm, I ran predictions and calculated the accuracy score using different levels of confidence. All scores were found to be over 70%, meaning seven out of ten times, the model was managing to correctly identify comments as either positive or negative. This is a satisfactory level of accuracy, but the results may be improved by training the model on more data.
